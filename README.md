BASICS OF PYTHONS IN THIS REPOSITORY.

# NUMPY LIBRARY

🔷 NUMPY (Numerical Python)

📌 What is NumPy?

NumPy is a fundamental library for numerical computing in Python.
It provides fast, memory-efficient arrays and mathematical operations.

# PANDAS LIBRARY

🔷 PANDAS (Data Analysis)

📌 What is Pandas?
Pandas is built on NumPy and used for data manipulation, cleaning, and analysis.

-> Handles structured data.

-> Easy data cleaning.

-> SQL-like operations.

-> Works with CSV, Excel, SQL, JSON.

# SEABORN LIBRARY

🔷 SEABORN (Statistical Visualization)

📌 What is Seaborn?

-> Seaborn is a high-level visualization library built on Matplotlib.

-> Beautiful plots.

-> Less code.

-> Built-in statistical plots

-> Works directly with Pandas
